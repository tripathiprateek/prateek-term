localStorage — Session Store

window.localStorage · browser-side
WHATSerialized tab state: title, protocol, cwd, sshProfile, remoteUrl. Keyed per session.
FORMATJSON string. Written by buildSessionData(), read by restoreSession() on next launch.
WRITEbuildSessionData() on app close / tab close events in renderer.
READrestoreSession() on app launch. Drives tab recreation with saved cwd.
WHENWritten on close, read on open. Persists across reboots. Cleared on explicit session reset.
📋

debug.log + Archives

userData/logs/debug.log · debug.log.1 … .5
WHATApplication debug log. Numbered rolling archives: debug.log → debug.log.1 → debug.log.2 … up to .5.
FORMATPlain text. Timestamped lines. Rotation trigger: 50 MB size or 30 days age (both configurable).
WRITEmain.js continuous append. maybeRotateDebugLog() shifts archives and starts fresh file.
READDebug Log Viewer in renderer: current log + archive list. "Rotate Now" button triggers immediate rotation.
WHENWritten continuously while app runs. Rotation checked on each write. Archives shifted on trigger.
⚙️

SSH Profiles — settings.json

userData/settings.json · Electron userData
WHATSSH profiles: host, port, user, identityFile, remotePath, cloudflareAccess flag, themes, keybindings, log rotation settings.
FORMATJSON. Sections: profiles[], theme, keybindings, logRotation (sizeMB, ageDays, maxFiles).
WRITESettings UI in renderer via IPC → main. User edits themes, key bindings, profile details.
READssh-utils.js on SSH connect. SCP handler for remotePath. Settings UI to populate forms.
WHENRead on every SSH connect and SCP upload. Written on every settings save. Persists across launches.
🔧

node-pty State

In-memory · main process · per-tab
WHATPTY handles keyed by ptyId. Each holds: pty object, shell PID, cwd, dimensions (cols × rows).
FORMATIn-memory Map. Not persisted — re-created on session restore via createTab with saved cwd from localStorage.
WRITEpty:create IPC handler spawns new PTY, adds to map. pty:resize updates dimensions.
READpty:write looks up by ptyId to write data. pty:close kills process and removes entry.
WHENExists only while app is running. One entry per open terminal tab. Gone on app exit.
