Tabs & Local Terminals

Open multiple terminal sessions side-by-side in tabs. Each tab runs an independent PTY process. Local tabs start immediately — no configuration needed.

Prateek-Term
LOCAL dist
SSH router-01
~/projects $ ls -la
drwxr-xr-x 8 prateek staff 256 Mar 27 09:12 .
~/projects $

Multi-tab workflow

Every tab is a full PTY — resize, scroll, paste, and log independently.

  • 1Press ⌘T to open a new local terminal tab.
  • 2Click any tab to switch; the terminal resizes to fit automatically.
  • 3Middle-click a tab to close it.
  • 4Drag tabs to reorder them.

Connection Manager — ⌘K

All saved devices listed in the sidebar. Click any profile to load it into the form, edit fields, and save.

  • 1Press ⌘K or click the grid icon in the title bar.

Serial console

Prateek-Term auto-discovers all available serial ports on your Mac. Pick a port, set the baud rate, and connect — the terminal opens instantly.

  • 1Set Protocol → Serial in the connection form.
  • 2Choose your port from the dropdown — refreshed on every form open.
  • 3Set baud rate (default 115200), data bits, parity, and flow control.
  • 4Click Connect — a serial tab opens with the device prompt.

Colour-coded tags

Tags appear as coloured dots on profile list items. Click a tag in the sidebar filter strip to instantly narrow the profile list to only tagged devices.

  • 1Open a profile → find the Tags section.
  • 2Type a tag name and pick a colour — press Enter to add.
  • 3In the sidebar, click any tag pill to filter the profile list.
  • 4Click the tag again or choose "All" to clear the filter.

Define actions in the profile form

Each action has a short name (max 20 chars) and a script. The script can be multi-line — it's sent as a single bracketed-paste block so multi-step commands work correctly.

  • 1Open a profile → scroll to the Actions section.
  • 2Click + Add Action → enter name and script → Save Action.
  • 3Reorder via edit (✎) or remove with ✕.
  • 4Save the profile — actions are available immediately in open terminals.
📤

Import / Export — share your action library across profiles without exporting credentials. The exported JSON contains only name and script fields.

One-click command execution

Hover over Actions in the context menu to reveal the submenu. Click any action — the script is sent to the terminal immediately via bracketed paste.

  • 1Right-click anywhere inside the terminal pane.
  • 2Hover over Actions to open the submenu.
  • 3Click the action name — the script executes in the active shell.
🔄

Live sync: if you add or edit an action while a terminal tab is open, the context menu updates immediately — no need to reconnect.

Capture & save in seconds

Great for turning one-off commands into reusable actions without leaving the terminal.

  • 1Select text in the terminal (click-drag or double-click a word).
  • 2Right-click → Add as Action…
  • 3Enter a short name (max 20 chars) in the mini-dialog → Save.

SSH Config — compatible with ~/.ssh/config, Termius, and Tabby.
JSON — full backup including all profile fields and actions.

  • 1Go to Settings → Import/Export section.
  • 2Import merges new hosts; existing profiles are never overwritten.

Actions Import / Export

Export just the actions for a profile as a clean JSON file (no passwords, no keys). Import merges by name — duplicates are skipped.

  • 1Open a profile → Actions section → click Export.
  • 2Open another profile → click Import → select the file.
  • 3New actions are appended; same-named actions are skipped.
